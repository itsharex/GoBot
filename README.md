## Build

1、Build Environment

Node.js

npm

Go

Wails link ：go install github.com/wailsapp/wails/v2/cmd/wails@latest

2、Pull and Build
git clone https://github.com/knowwhythat/GoBot.git

cd GoBot

wails build
